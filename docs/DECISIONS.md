2026-07-13 — Issue #20 robots.txt: enforced minimally, not policy-only. Why: stated compliance must match actual behavior — documenting a permissive posture while not checking robots.txt at all would be a claim we can't back up. The website sub-workflow fetches `/robots.txt` first (5s timeout, no retry), parses only the `User-agent: *` block's `Disallow` paths, and prefix-matches each of home/about/pricing (and their fallback paths) against them before fetching.
